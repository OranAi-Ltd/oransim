The Causal Stack

Counterfactual reasoning, first-class.

Oransim is built causal-first — treatment effect estimation, DAG-aware attention, and intervention-aware diffusion are core interfaces, not afterthoughts. Every component cites the recent literature it builds on.

🧠

Causal Transformer World Model

6-layer multi-head self-attention with explicit treatment/covariate/outcome factorization, DAG-aware attention bias, per-arm counterfactual heads, and a representation-balancing loss. Based on CaT (Melnychuk ICML'22), CausalDAG-Transformer, BCAUSS, TARNet/Dragonnet, CInA (Arik NeurIPS'23).

primary
⚡

Causal Neural Hawkes Process

Transformer-parameterized temporal point process for 14-day diffusion with treatment vs. control event typing and intervention-aware intensity. Based on Mei & Eisner (NeurIPS'17), Zuo et al. (ICML'20), Geng et al. (NeurIPS'22 counterfactual TPP).

primary
🌐

64-node Pearl SCM

Hand-designed structural causal model over the marketing funnel (117 edges). Pearl's three-step do() counterfactual evaluation (abduction → action → prediction) with mediators for group discourse (Sunstein 2017) and cascades.

👥

1M IPF Agents + 10k LLM Personas

Iterative Proportional Fitting (IPF / Deming-Stephan 1940) calibrates 1M virtual consumers to published demographic priors. Top-10k salient agents get LLM-powered personas for qualitative feedback + simulated group chats (Sunstein 2017 polarization).

🧪

LightGBM Quantile  /  Parametric Hawkes

Zero-dependency baselines retained for sub-millisecond inference and OrancBench ablations. Three P35/P50/P65 quantile regressors per KPI (Koenker 2005 · Ke et al. 2017); classical exponential-kernel Hawkes (Hawkes 1971) with Ogata (1981) thinning.

fast baselines
🧩

Platform × DataProvider

Two-axis extensibility. XHS / RedNote is the v1 reference implementation; TikTok, Instagram, YouTube Shorts, Douyin ship as day-1 stubs with concrete roadmap. Each platform accepts any DataProvider (Synthetic / CSV / JSON / OpenAPI / your own).

Architecture

One prediction pass, seven engines.

A typical prediction: Creative + Budget → PlatformAdapter (+DataProvider) → Causal Transformer World Model (factual + per-arm counterfactual quantiles) + Agent Layer → Pearl SCM counterfactual evaluation → Causal Neural Hawkes 14-day diffusion → structured JSON across 14-19 schemas.

Quickstart

Clone, install, predict — in 60 seconds.

v0.1.0-alpha ships the full skeleton — flagship docs, Causal Transformer and Causal Neural Hawkes code (weights arrive in v0.2), five platform scaffolds, and the synthetic data generator to train locally.

pip install 'oransim[ml]' unlocks the research-grade PyTorch models; plain pip install oransim keeps the LightGBM + parametric Hawkes baselines usable.

Full Docs → 
# 1. Clone and install
git clone https://github.com/ORAN-cgsj/oransim.git
cd oransim
pip install -e '.[ml,dev]'

# 2. Generate synthetic training data
python -m backend.scripts.gen_synthetic_data \
    --out data/synthetic --seed 42

# 3. Train the Causal Transformer
python -m backend.scripts.train_transformer_wm \
    --data data/synthetic/scenarios_v0_1.jsonl \
    --epochs 50

# 4. Or just inspect the architecture
python -c "
from oransim.world_model import get_world_model
wm = get_world_model('causal_transformer')
print(wm.describe())
"

Four models. One registry. Zero lock-in.

Each model ships architecture + training loop + inference code today. Pretrained weights start shipping with v0.2. OranAI Enterprise Edition trains the same architectures on proprietary real data.

CausalNeuralHawkesProcess
Primary diffusion
≈ 900k params · 4 × 128-dim · PyTorch

LightGBMQuantileWorldModel
Fast baseline
Sub-millisecond CPU · zero-GPU · PCA feature projection
Refs: LightGBM (Ke et al. 2017) · Quantile Regression (Koenker 2005)
ParametricHawkes
Classical baseline
Closed-form intensity · Ogata thinning · pure-Python
Refs: Hawkes (1971) · Ogata (1981) · Laub, Taimre, Pollett (2015)
