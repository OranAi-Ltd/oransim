Abstract

Marketing prediction has long oscillated between academic simulators that don't ship and enterprise tools that don't explain. Oransim is the first open-source framework to make counterfactual causal reasoning a first-class interface — per-arm counterfactual heads, DAG-aware attention, representation-balancing regularizers, and intervention-aware temporal point processes — while remaining pip install-shaped and deployable behind a single FastAPI service. The result is a digital twin that predicts, explains, and runs what-ifs — for every platform adapter the community plugs in.

01 · The causal stack

Six engines, one contract.

Each component is backed by a recent peer-reviewed paper and cites its lineage in both the code and the docstrings. The primary models ship architecture + training + inference today; pretrained weights land with v0.2.

primary WM
01.A

Causal Transformer · world model

Six-layer multi-head self-attention with explicit treatment / covariate / outcome token factorization, DAG-aware attention bias over the Pearl SCM, per-arm counterfactual heads, and a representation-balancing regularizer (HSIC or adversarial IPTW).

Refs.　CaT (Melnychuk ICML'22 · arXiv:2204.07258) · CausalDAG-Transformer (Zhang'23) · TARNet (Shalit ICML'17) · Dragonnet (Shi NeurIPS'19) · BCAUSS (Tesei'21) · CInA (Arik NeurIPS'23) · HSIC (Gretton'05).
primary diffusion
01.B

Causal Neural Hawkes · diffusion

Transformer-parameterized neural temporal point process for 14-day cascade forecasting, with organic vs. paid-boost event typing and intervention-aware intensity enabling do() queries on cascades (e.g. "stop boosting on day 3").

Refs.　Neural Hawkes (Mei & Eisner NeurIPS'17) · Transformer Hawkes (Zuo ICML'20) · Intensity-Free TPP (Shchur ICLR'20) · Neural STPP (Chen ICLR'21) · Counterfactual TPP (Geng NeurIPS'22) · Ogata thinning (1981).
01.C

Pearl SCM · 64 ⟶ 117

Hand-designed DAG over the full marketing funnel, with mediators for group discourse (Sunstein 2017) and cascades. Three-step counterfactual evaluation.

01.D

1M IPF agents + 10k LLM personas

Iterative Proportional Fitting (Deming-Stephan 1940) calibrates 1M consumers to demographic priors. Top-10k salient agents become LLM-driven personas for qualitative feedback.

fast baselines
01.E

LightGBM · parametric Hawkes

Zero-dependency fallbacks. Three P35/P50/P65 quantile regressors per KPI (Ke'17 · Koenker'05); exponential-kernel Hawkes (1971) with Ogata thinning. Sub-millisecond CPU inference.

01.F

Platform × DataProvider · two axes

Same PlatformAdapter interface across XHS (v1), TikTok, Instagram, YouTube Shorts, Douyin (stubs). Each platform accepts a pluggable DataProvider — Synthetic, CSV, JSON, OpenAPI, or your own — so a single platform can draw from multiple data sources without changing the platform logic.

01.G

Soul agent orchestration · deduped

In-flight request coalescing (leader / followers pattern) collapses duplicate LLM calls before they hit the provider. Persona-card caching; configurable SOUL_POOL_N; v0.5 roadmap scales to 10k agents on a Ray cluster.

02 · Architecture

One prediction pass,
seven engines.

Creative + Budget → PlatformAdapter (pulls data via pluggable DataProvider) → Causal Transformer World Model (factual + per-arm counterfactual quantile predictions) + Agent Layer (1M IPF + 10k LLM personas) → Causal Engine (64-node Pearl SCM + three-step do()) → Causal Neural Hawkes (14-day diffusion with intervention rollout) → Prediction JSON across 14–19 schemas. LightGBM + parametric Hawkes remain available through the registry as fast baselines.

The seven-engine pipeline is fully composable — any engine can be swapped via the registry. A typical prediction takes 60 s on a single machine with LLM_MODE=api.

05 · Quickstart

Clone, install, predict.

v0.1.0-alpha ships the full skeleton — Causal Transformer and Causal Neural Hawkes code, five platform scaffolds, and a synthetic data generator for local training.

Pick the install flavour that matches your appetite for dependencies:

pip install oransim — LightGBM + parametric Hawkes baselines only, no PyTorch.

pip install 'oransim[ml]' — unlocks Causal Transformer + Causal Neural Hawkes.

pip install 'oransim[ml,dev,docs]' — everything, for contributors.

⟶ Full docs
oransim — quickstart.sh
———
# 1 · Clone and install
git clone https://github.com/ORAN-cgsj/oransim.git
cd oransim
pip install -e '.[ml,dev]'

# 2 · Generate synthetic training data
python -m backend.scripts.gen_synthetic_data \
    --out data/synthetic --seed 42

# 3 · Train the Causal Transformer
python -m backend.scripts.train_transformer_wm \
    --data data/synthetic/scenarios_v0_1.jsonl \
    --epochs 50

# 4 · Inspect the architecture
python -c "
from oransim.world_model import get_world_model
wm = get_world_model('causal_transformer')
print(wm.describe())
"
